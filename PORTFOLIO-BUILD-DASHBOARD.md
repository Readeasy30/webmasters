## Daily build order

Use this order so work stays productive:

1. Pick one website focus.
2. Pick one traffic goal.
3. Pick one visitor action.
4. Create one useful content asset.
5. Create one matching Canva/social asset.
6. Publish or prepare the asset.
7. Record the action in the posting log.
8. Move to the next useful task.
