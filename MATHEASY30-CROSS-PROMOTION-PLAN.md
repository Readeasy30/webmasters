# MathEasy30 Cross-Promotion Plan

Last updated: 2026-05-29

## Purpose

Use this plan to grow MathEasy30 through ReadEasy30 as a calm sister practice site.

The first goal is simple:

**When someone likes the calm ReadEasy30 practice style, show them MathEasy30 as the same kind of daily confidence path for math.**

## Shared Education Family Message

ReadEasy30 helps with reading practice.

MathEasy30 helps with math practice.

Both sites should feel:

- calm
- simple
- patient
- daily
- no shame
- confidence-building
- good for helpers, parents, adults, and older learners

## Core MathEasy30 Message

Practice math one small step at a time.

No rush. No shame. Just steady progress.

## Best Cross-Promotion Placement

### On ReadEasy30

Add soft mentions where natural:

- homepage lower section
- completion screen
- parent/helper tip area
- reading levels page
- calm reading routine page
- footer or sister-site card

Suggested card:

```md
## Need math practice too?

Try MathEasy30, our calm sister site for daily math practice.

Small steps. Simple problems. Steady confidence.

[Open MathEasy30]
```

### On MathEasy30

Add ReadEasy30 as the sister reading site:

```md
## Need reading practice too?

Try ReadEasy30, our calm sister site for daily reading practice.

Short lessons. Simple questions. Patient progress.

[Open ReadEasy30]
```

## Do Not Overdo It

Do not make every page a cross-sell.

Use one clear sister-site mention where it helps the visitor.

## Cross-Promotion Rules

Use:

- calm practice
- small steps
- daily routine
- confidence
- helper support
- no shame
- move at your own speed

Avoid:

- guaranteed improvement
- official grade placement
- medical or diagnostic claims
- saying the site replaces a teacher or tutor
- shame language like “bad at math”
- pressure language like “catch up fast”
