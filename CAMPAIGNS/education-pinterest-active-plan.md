# Education Pinterest Active Plan

Purpose: use Pinterest as a simple evergreen traffic source for ReadEasy30 and MathEasy30 worksheets, daily practice pages, and support guides.

## Pinterest job

Pinterest should send people to useful pages that solve one clear problem:

- printable reading worksheets
- printable math worksheets
- calm reading practice
- adult reading practice
- ESL reading practice
- math practice without pressure
- parent/helper tips

## Pin design rules

- Use vertical format: 1000 x 1500 or similar.
- Large readable headline.
- One main message per pin.
- Include site name on image.
- Use calm colors.
- Avoid clutter.
- Avoid tiny worksheet screenshots.
- Use adult-friendly design, not childish design.

## Pin description template

Use this structure:

1. Problem sentence
2. What the page gives
3. Who it helps
4. Link/CTA

Example:

Need simple reading practice away from the screen? ReadEasy30 has free printable reading worksheets with short passages, vocabulary help, simple questions, and helper tips. Good for parents, tutors, adult learners, and ESL learners. Open the free worksheet hub.

## Review after 14 days

Record:

- Which board got the most saves?
- Which pin got the most outbound clicks?
- Did reading or math perform better?
- Did worksheets perform better than app pages?
- Which headline should be reused?

## Decision rule

If worksheet pins outperform app pins, build more worksheet pages and pins before promoting broader app pages.
