# Email 30-Day Quota Plan

Last updated: 2026-06-04

Campaign: **ReadEasy30 + MathEasy30**

Sender / reply email:

```text
free@readeasy30.com
```

Main campaign link:

```text
https://readeasy30.com/free-reading-and-math-practice.html
```

## Goal

Run email outreach every day for 30 days without rebuilding the plan.

Daily operating target:

```text
240 qualified outreach actions per day
```

Daily sending split after setup:

```text
200 emails/day through Mailjet free tier
40 contact-form or manual one-to-one actions/day
```

## Why Mailjet first

Mailjet is the first planned email provider because its free plan supports:

```text
6,000 emails/month
200 emails/day
```

That covers most of the daily quota.

## Why the last 40 are manual/forms

The daily outreach target is 240. Mailjet free only covers 200 per day.

So the remaining 40 daily actions are handled as:

```text
contact forms
manual one-to-one Gmail messages
held for review
blocked/no contact route
```

## Before sending from Mailjet

Complete these items first:

1. Confirm `free@readeasy30.com` can receive email.
2. Confirm `free@readeasy30.com` can send email.
3. Create Mailjet account.
4. Add sender/domain verification.
5. Add required domain records shown by Mailjet.
6. Create suppression/opt-out tracker.
7. Use only verified public organization contacts.
8. Do not import scraped, purchased, or private lists.

## Compliance rules

Every direct email must include:

- accurate sender identity
- truthful subject line
- clear free-resource message
- campaign website link
- ReadEasy30 and MathEasy30 links
- opt-out/remove line
- sender contact email
- no misleading claims

## Opt-out line

Use this line in direct emails:

```text
If this is not useful or you prefer not to receive another message from me, reply “remove” and I will not contact you again.
```
