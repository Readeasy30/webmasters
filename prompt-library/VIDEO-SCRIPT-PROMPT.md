# Video Script Prompt

Use this prompt to create short website videos, social videos, or simple YouTube scripts.

## Master prompt

Create a video script for this website:

Website: [website name and domain]

Audience: [primary viewer]

Video type: [homepage intro / short social video / YouTube explainer / tutorial]

Length: [15 seconds / 30 seconds / 60 seconds / 3 minutes]

Main message: [one clear promise]

Next action: [where the viewer should go]

Rules:

- Use simple language.
- Use short sentences.
- Make it understandable without sound.
- Include captions/on-screen text ideas.
- Avoid unsupported claims.
- Avoid hype.
- Do not add affiliate, ad, tracking, or payment claims unless approved.

Create:

1. Hook
2. Voiceover script
3. On-screen text
4. Visual scene ideas
5. Call-to-action
6. Safer alternate wording for any risky claims

## 30-second structure

0-3 seconds: Hook

4-12 seconds: What the site helps with

13-23 seconds: Examples or benefits

24-30 seconds: Website name and next action
