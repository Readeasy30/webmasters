# Avatar Prompt

Use this prompt to create a site-specific avatar plan.

## Master prompt

Create an avatar plan for this website:

Website: [website name and domain]

Audience: [primary visitors]

Main goal: [what the avatar should help visitors do]

Tone: [friendly / calm / fun / local / businesslike / etc.]

The avatar must:

- speak in short, clear sentences
- explain the site quickly
- guide visitors to the next useful action
- avoid hype
- avoid risky claims
- avoid asking for private information
- work even if visitors skip the video

Create:

1. Avatar name ideas
2. Avatar role
3. Short personality description
4. Homepage intro script under 40 seconds
5. Three quick helper messages
6. Best page placement
7. Mobile safety notes
8. What not to say

## Site-specific notes

### ReadEasy30

Keep it calm, no-shame, slow, and encouraging. Bubbles is a reading coach, not a tutor who scolds.

### PetNeeds.ai

Helpful pet care information only. Do not replace a veterinarian.

### SlotsFreeUSA

Adult-focused, fun, transparent, and responsible-play aware.

Friendly Branson trip buddy. Help visitors choose shows, food, attractions, lake ideas, and local deals.

### RestaurantAIbot / Restaurants.ai

Help restaurant owners understand how the website or AI service can bring more calls, orders, and customer follow-up.
