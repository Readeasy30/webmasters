# Canva Video Checklist

Use this before making website intro videos, Shorts, Reels, Pins, or YouTube videos in Canva.

## Before creating

- Pick one website.
- Pick one audience.
- Pick one main message.
- Pick one next action.
- Keep the first version short.

## Canva setup

Recommended formats:

- Vertical short: 1080 x 1920
- YouTube thumbnail: 1280 x 720
- Website video cover: 1920 x 1080
- Pinterest pin video: 1000 x 1500 or vertical short format

## Video structure

1. Opening title
2. One clear problem or promise
3. Two or three quick benefits
4. Website name
5. One next action

## Design rules

- Big readable text.
- High contrast.
- One message per screen.
- Avoid clutter.
- Use the same colors and tone as the website.
- Make it understandable without sound.
- Add captions or on-screen text.

## Export rules

- Keep filenames clear.
- Store scripts and notes in GitHub.
- Store final video in Canva, YouTube, Cloudflare Stream, or another approved platform.
- Do not commit large video files to GitHub unless there is a clear reason.

## Review before publishing

- Does the video match the site promise?
- Is the next action obvious?
- Is any claim risky or unsupported?
- Is the video too long?
- Does it work on mobile?
- Is the call-to-action safe and honest?
