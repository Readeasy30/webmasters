# Social Post Templates

Reusable social templates for Wholelychit websites.

## Goal

Turn each useful website idea into simple posts for multiple channels.

Keep the tone clear, helpful, and low-pressure.

## Core Rules

- One post, one idea.
- Lead with the reader problem.
- Use plain language.
- Do not hype results.
- Do not spam communities.
- Match the platform.
- Send people to one useful next step.

## Universal Post Formula

```md
Hook:
Problem:
Simple help:
Next step:
Link/CTA:
```

## Project-Specific Hooks

### ReadEasy30

- Reading practice should not feel like punishment.
- A calm reader learns better than a pressured reader.
- Start with one short story and three questions.
- Confidence comes before speed.
- If a child freezes while reading, slow down.

### MathEasy30

- Math practice works better when it feels safe.
- Start small. Win one problem at a time.
- A short daily habit beats a long stressful session.
- When a learner freezes, review one step back.
- Math confidence is built in small wins.

### PetNeeds.ai

- New pet? Start with the basics first.
- Not every pet question needs panic, but some need a vet.
- A simple pet routine helps the whole family.
- Before buying more pet products, check what the pet truly needs.
- Safe pet care starts with clear questions.

### Restaurants.ai

- Customers choose faster when answers are easy to find.
- A restaurant website should answer simple questions fast.
- Menus, hours, location, parking, and specials should be easy on mobile.
- AI can help restaurants answer repeat questions.
- Local trust starts before the customer walks in.

### SlotsFreeUSA

- Learn slot terms before spending money.
- Free slots are entertainment, not income.
- Play for fun. Set limits. Walk away.
- Beginners should understand volatility before real-money play.
- Never chase losses.

## CTA Bank

Use low-pressure CTAs:

- Start here.
- Read the beginner guide.
- Save this checklist.
- Try Day 1.
- Compare your options.
- Learn the basics first.
- Come back tomorrow for the next step.
