# Portfolio Weekly Execution Board

Last updated: 2026-06-02

This file turns the marketing plans into weekly action. Use it to keep work moving without jumping between too many projects.

## Rule

Each week gets:

1. One main project focus
2. One visitor problem
3. One website page or article
4. One short video script
5. Three social posts
6. One Canva asset prompt
7. One simple next action

Website code stays in each website repo. Marketing strategy and reusable campaign assets stay here.
