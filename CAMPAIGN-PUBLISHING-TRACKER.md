# Campaign Publishing Tracker

Use this file to track what has been prepared, published, improved, or blocked across Wholelychit projects.

## Rule

Track manually first. Do not add live analytics, tracking scripts, paid tools, bulk posting, or automated reporting until approved.

## Status Labels

| Label | Meaning |
|---|---|
| Planned | Idea is approved for future work |
| Drafted | Copy or script exists |
| Designed | Canva or visual prompt is ready |
| Exported | Upload-ready asset exists |
| Published | Live on website or social channel |
| Improved | Existing item was updated |
| Blocked | Needs access, account setup, approval, or manual external action |
| Paused | Intentionally stopped for now |
| Needs edit | Asset exists but should be fixed before use |
