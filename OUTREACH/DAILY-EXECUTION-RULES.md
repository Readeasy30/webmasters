# Daily Execution Rules

Last updated: 2026-06-02

This file keeps outreach moving without random jumping or repeated work.

## Daily outreach target

Complete 5 safe contact-form submissions per active outreach day.

Do not mass-send. Do not bulk-blast. Do not scrape private emails.

## Daily order

1. Pick one segment for the day.
2. Open `EDUCATION-PROSPECT-QUEUE.md`.
3. Choose the next 5 prospects marked `Contact-form only` or `Direct verified public email`.
4. Verify the organization page still loads.
5. Use the matching script from `CONTACT-FORM-SCRIPTS.md`.
6. Record submitted date.
7. Set one follow-up date 7 to 10 days later.
8. Stop on opt-out, no-contact signal, or mismatch.

## Segment rotation

Use this weekly pattern:

| Day | Segment |
|---|---|
| Monday | Libraries / adult literacy / ESL |
| Tuesday | Head Start / early learning / parent education |
| Wednesday | Adult education / GED / workforce |
| Thursday | Tutors / homeschool / after-school |
| Friday | Churches / community nonprofits / reentry support |

## Follow-up rule

One follow-up only.

Do not keep sending after no reply.

## Blocker rule

If a tool blocks sending or drafting, do not keep retrying.

Record the block and move to contact forms or the next safe prospect.

## Tracking columns

Each prospect should track:

- Organization
- Segment
- Website
- Contact route
- Contact type
- Message version
- Submitted date
- Follow-up date
- Reply status
- Opt-out/no-contact status
- Notes

## Daily done definition

A daily outreach block is complete when one of these is true:

- 5 safe contact forms submitted and logged.
- All available prospects in that segment are exhausted.
- A real blocker occurs and is recorded.

## Real blockers

Stop only for:

- missing permissions
- account sign-in problem
- credentials/private keys
- billing
- live ads or tracking
- framework migration
- major code deletion
- unclear production risk

Otherwise continue to the next safe prospect or next safe marketing task.
